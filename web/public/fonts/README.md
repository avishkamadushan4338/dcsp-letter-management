Place `NotoSansSinhala-Regular.woff2` in this folder.

Download it from Google Fonts (Noto Sans Sinhala) and convert/export as
`.woff2`, or grab a pre-built woff2 from a font CDN mirror. This binary isn't
checked into the scaffold — `css/sinhala.css` references
`fonts/NotoSansSinhala-Regular.woff2` and will silently fall back to
`Iskoola Pota` / system fonts until the file is added here.
